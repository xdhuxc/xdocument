### top 命令快捷键
在 Linux shell 下，输入 top 命令，快捷键使用如下：
按 ‘X’ 键，再按 ‘B’ 键，默认按 CPU 使用率升序排列
按住 ‘Ctrl’ 键，再按 ‘P’ 键，是按 CPU 使用率降序排列。
按住 ‘Ctrl’ 键，再按 ‘M’ 键，是按 内存 使用率降序排列。
按住 ‘Ctrl’ 键，再按 ‘T’ 键，是按 进程运行时间降序排列。
